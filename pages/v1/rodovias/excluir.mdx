# Excluir Rodovia
<p className="text-red-600">Disponível apenas para ambiente de desenvolvimento ⚠️</p>

Rota específica para exclusão de uma rodovia através do id.

### Rota

> /api/highways/**:id**

### Parâmetros

- `:id`
  - ID da rodovia

### Requisição de exemplo

```sh
curl -L \
  -X DELETE \
  https://brazilian-highways-api.vercel.app/api/highways/1
```

### Resposta de exemplo
```json
// status 200
```
