# Excluir Rodovia

Rota específica para exclusão de uma rodovia através do id.

### Rota

> /api/highways/:id

### Parâmetros

Não há parâmetros para a rota.


### Requisição de exemplo

```sh
curl -L \
  -X DELETE \
  https://brazilian-highways-api.vercel.app/api/highways/1
```

### Resposta de exemplo
```json
// status 200
```
