# Listar Rodovias

Listar todas as rodovias da aplicação incluindo pesquisa com base em parâmetros opcionais.

### Rota

> /api/highways

### Parâmetros

- `q=<string>`
  - Opcional
  - Pesquisa entre os campos ***name*** e ***code***


### Requisição de exemplo

```sh
curl -L \
  -X GET \
  https://brazilian-highways-api.vercel.app/api/highways
```
