# Busca de Marcações  por Rodovia

Busca de marcações de quilometragens por rodovia incluindo pesquisa através de parâmetros.

### Rota

> /api/highways/:highway_id/keypointers

### Parâmetros

- `q=<string>`
  - Opcional
  - Pesquisa entre os campos ***km***, ***city*** e ***uf***

### Requisição de exemplo

```sh
curl -L \
  -X GET \
  https://brazilian-highways-api.vercel.app/api/highways/2/keypointers
```
