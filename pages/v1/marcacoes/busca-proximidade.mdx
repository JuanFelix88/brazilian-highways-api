# Busca de Marcações por Proximidade

Busca de marcações de quilometragens por proximidade de uma determinada posição.

### Rota

> /api/keypointers

### Parâmetros

- `position=<number,number>`
  - Posição alvo da pesquisa
- `ray_proximity=<number>`
  - Opcional
  - Padrão: **1500**
  - Define o raio para buscas em proximidade
  - Medida em **metros**

### Requisição de exemplo

```sh
curl -L \
  -X GET \
  https://brazilian-highways-api.vercel.app/api/keypointers?ray_proximity=1200&position=-25.53613924132,-48.556601830
```
